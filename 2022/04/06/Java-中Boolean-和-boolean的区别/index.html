boolean 和 Boolean 的区别

boolean 是基础数据类型，而 Boolean 是它的封装类。

boolean 创建的对象的值默认是 false，而 Boolean 创建的对象在没有实例化之前是 null.

总结

在大多数情况下，使用 boolean 就已经足够。
